# Next.js custom server with Fastify

Setup for Next.js custom server using Fastify.

## Get started

**Install:**

```bash
pnpm install
```

**Build and start:**

```bash
pnpm start
```

**Start test mode using Vitest:**

```bash
pnpm test
```

**Start test mode with coverage:**

```bash
pnpm coverage
```

**Build:**

```bash
pnpm build
```
